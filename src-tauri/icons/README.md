# Icons Directory

This directory should contain the following icon files for Tauri:

- `32x32.png` - 32x32 pixel PNG icon
- `128x128.png` - 128x128 pixel PNG icon
- `128x128@2x.png` - 256x256 pixel PNG icon (for retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

You can generate these icons using tools like:
- [Tauri Icon Generator](https://github.com/tauri-apps/tauri-icon)
- Online icon converters
- Image editing software

For development, the app will work without these icons, but they're required for production builds.
